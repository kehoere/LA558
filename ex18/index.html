Exercise 18: D3

non-map D3 visual: Force Directed Graph

I like how the map is interactive by dragging the colored circles and how the visual stays intact after altering the positon of the colored circles.

For the D3 code, you can change the function of the dragged circle positions, the color and shape of the circles, and add in different data to be graphed. You would need to know the colors, width of the lines, and scale of the lines on the graph.


D3 map visual: Bubble Map

I like how the bubbles on the map depict the population density. It is a nice visual respresentation showing which counties have more people and where the population is concentrated in the United States.

The code can be altered based on width, height and radius of the map and circles. You are also able to adjust the colors of the circles and import different data for the population.


one other D3 example: Hierarchical Edge Bundling

I like how the diagram shows line connections between the words. It is a really powerful visual diagram with the varying denses of the overlapping lines.

The code mainly has imported data, line characteristics, and context of how the lines are displayed. You would need to know the width, radius, and curvature dimension of the lines, as well as the data information you would want to use.
